2. hexo -d失败

   2. 通过nvm进行版本降级
      1. brew 安装
      2. 安装前先update brew版本
   3. `brew install nvm`
   4. `source ~/.nvm/nvm.sh`
   5. `nvm install 13.14.0`
   6. `nvm use 13.14.0`
